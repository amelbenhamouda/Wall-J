Wall-J
=======

Wall-J : the Space Cleaner is a directed game object written in JAVA.
This game aims to clean the game area of present waste by pushing them with bombs
into the trashcans located over the screen.

Utilisation
===========

## Compilation
	* ant compile

## Exécutable
    * ant jar
    * java -jar wallj.jar

## Documentation
	* ant javadoc
